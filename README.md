## 🧩 **Easy Plugin Management System**

### How It Works
All plugins are now managed from a **single JavaScript array** in `script.js`:

```javascript
const plugins = [
  {
    name: 'Plugin Name',
    description: 'What it does',
    installCmd: 'npm install command'
  },
  // Add more...
];
```

### Current Default Plugins (6)
1. Data Processor
2. Image Analyzer
3. Text Generator
4. API Connector
5. ML Pipeline *(new)*
6. Data Visualizer *(new)*

### To Add a New Plugin
Simply add one object to the `plugins` array - that's it! No HTML changes needed.

**See `PLUGINS.md` for detailed instructions.**
